Comment ça marche ?

TF-IDF + Cosinus

Calcule l'importance de chaque mot (TF-IDF) puis mesure l'angle entre les deux vecteurs de texte. Plus l'angle est petit, plus les textes sont similaires.

Exemple :
"chat" apparaît 3× dans A, 1× dans B → poids élevé

Indice de Jaccard

Compare les ensembles de mots uniques : intersection / union. Un score de 1 = mêmes mots exacts, 0 = aucun mot en commun.

Exemple :
A={chat,souris,jardin} B={chat,souris,cour} → 2/4 = 50%

N-gram Overlap

Détecte les séquences consécutives de mots (bigrams) identiques entre les deux textes. Capture les phrases copiées mot pour mot.

Exemple :
A="le chat noir" B="le chat noir" → bigram "le chat" match

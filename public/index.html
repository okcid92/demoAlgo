Comment ça marche ?

TF-IDF + Cosinus

Calcule l'importance de chaque mot (TF-IDF) puis mesure l'angle entre les deux vecteurs de texte. Plus l'angle est petit, plus les textes sont similaires.

Exemple :
"chat" apparait 3x dans A, 1x dans B -- poids eleve

Indice de Jaccard

Compare les ensembles de mots uniques : intersection / union. Un score de 1 = memes mots exacts, 0 = aucun mot en commun.

Exemple :
A={chat,souris,jardin} B={chat,souris,cour} -- 2/4 = 50%

N-gram Overlap

Detecte les sequences consecutives de mots (bigrams) identiques entre les deux textes. Capture les phrases copiees mot pour mot.

Exemple :
A="le chat noir" B="le chat noir" -- bigram "le chat" match

Shingling (Winnowing)

Genere des empreintes (hash) de sequences de mots et les compare. Detecte les copies meme si les paragraphes sont reordonnes.

Exemple :
Shingle = hash(mot1..mot5) -- fenetre glissante retient le min

Semantique (BERT)

Transforme le texte en vecteur dense capturant le sens profond. Detecte les paraphrases et traductions que les algorithmes lexicaux manquent.

Exemple :
"voiture rapide" ~= "automobile veloce" -- score 0.91

SimHash (LSH)

Genere un hash 64 bits qui preserve la similarite. Deux textes proches ont des hashes proches (distance de Hamming faible).

Exemple :
Hamming(A, B) = 8 bits differents sur 64 -- similarite 87.5%

LCS

Trouve la plus longue sous-sequence commune (pas forcement contigue). Detecte les suppressions et insertions de mots ("plagiat chirurgical").

Exemple :
A="a b c d e" B="a c d e" -- LCS = 4 -- Dice = 80%
